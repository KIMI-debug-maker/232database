# 金融学理论

## 1. 导论

### 1.1. 金融学定义

+ 金融学是研究人们在不确定条件下如何进行资源跨期配置的科学。
+ 金融决策两个特性：
  + 成本和收益是在时间上分布的
  + 决策者（或者任何人）无法预先明确知道结果。e.g. 五道口枣糕王和一个自信投资因为非典的失败

+ 运筹学已有结果，是对一个问题的优化和分配，与金融有区别。
+ 金融学属于哲学的范畴，而哲学是“是一门给人智慧、使人聪明的学问”，强调的是“思辨”过程，因此金融学的基本原理是关于“钱”的问题的思辨

### 1.2. 金融理论与作用

+ 金融理论由一系列**概念**和**定量模型**组成。
  + 概念：帮助在时间上配置资源
  + 定量模型：帮助估值、制定决策、执行决策
+ 金融的基本信条：金融的最终功能是为了满足人们的消费偏好。金融系统中各个机构的存在正是为了实现这个最终目标
+ 帕累托改进？金融既是也不全是
  + 帕累托最优：在不伤害别人的情况下达到最好
  + 金融在每个时间点可能实现了帕累托，但实际上损害了一部分人利益

### 1.3. 为什么研究金融

+ 管理个人资源：代理问题
+ 处理商务世界中的问题：例如商务中出现金融概念、技术、术语
+ 寻求令人感兴趣和回报丰厚的职业
+ 公共选择中发挥个人作用
  + 数年前奥巴马VS罗姆尼总统之争，医改问题。立场问题，穷人还是富人？
  + 经济适用房问题反思。经济学家：”经济适用房不应该配厕所“，因为有钱人会抢走。e.g. 真正的灾民不会在乎饭里有沙子
+ 扩展思路
  + 股改：股权分制(为了防止被对手获取，让部分国有股无法流通)->股票全流通(流动性冲击，向中小股东补偿)

### 2.2. 货币的时间价值

+ 对钱来说，时间值钱吗？

  + 今天的1万元，1年后的1万元，更喜欢哪个？肯定喜欢今天的

+ 货币的时间价值：今天的单位货币总比明天的单位货币值钱——金融学第一要义

  + 怎么解释负利率：市场摩擦，流动性问题。当市场没有好机会的时候，需要花成本存钱

+ **现值与贴现的概念**

  + 现值（Present Value）：是未来的现金流在今天的价值
  + 贴现：就是把未来的收益换算成今天的价值的过程
    + 你愿意出多少钱购买未来的1元钱呢？换句话说，今天的多少钱和未来的1元钱是等值的呢？
  + 未来1单位货币现在的价值，表示为收益率的形式$PV(1单位未来货币) = \frac{1}{1+r}$
    + $r$通常叫做贴现率(discount rate)，也被叫做资金的机会成本（opportunity cost of capital)
  + 机会成本：投A损失B，逝去的部分

+ **单笔现值与净现值的计算**

  + 未来一年的单位货币相当于现在的价值$\frac{1}{1+r}$， 所以对未来一年的CF的现金流，贴现到现在的价值为$PV(CF) = \frac{CF}{1+r}$
  + 多期现金流：$PV = \sum_{t=1}^n\frac{CF_t}{(1+r)^t}$

  <img src="images\image-20230906190506635.png" alt="image-20230906190506635" style="zoom:50%;" />

  + 现值不是凭空而来，是投资带来的回报。净现值等于现值减必要的投资$NPV = \sum_{t=1}^n\frac{CF_t}{(1+r)^t}-I_0$
  + 净现值法则
    + NPV>0，增值，超过机会成本的回报
    + NPV=0，保值，和机会成本回报相等
    + NPV<0，贬值，低于机会成本的回报

### 2.3. 商业应用

+ 年金计算
  + 贷款1,000,000，贷款年利率7.05%，贷款期限10年，每月还款一样，还款额是多少？ PV=1000000，N=120, r = 7.05%/12, 求CF即可
  + 按月递减还款：对银行来说“按月递减还款”所收利息更少，为什么银行还愿意提供？按月递减，根据PV公式确实利率更低，但因为实际最后PV和等额的PV是一样的，所以没区别
  + 等额还款：对于贷款人来说，“等额还款”所支付利息更高，为什么还有人选择？取决于机会成本
    + 如果贷款人机会成本等于银行贷款利息，那么“等额还款”和“按月递减还款”的现值相等，因此两者无差异
    + 如果贷款人机会成本小于银行贷款利息，那么“等额还款”使得贷款人付出的现值更大，因此贷款人选择“按月递减还款”更合适
    + 如果贷款人机会成本大于银行贷款利息，那么“等额还款”更合适
+ **被忽视的利率**：看似很低的日利率，会产生滚雪球的巨大年利率；固定利息下每期还本金，看似合规但随着本金减少利息没有减少，实际年化利率更高

### 2.4. 现值计算的简化

+ 对有规律的现金流，可以采用公式计算
  + **永续年金（Perpetuity）**：每年固定支付，永不停止。$PV = \frac{C}{1+r}+\frac{C}{(1+r)^2}+...=\frac{C}{r}$
  + **增长型永续年金**：每年收的支付额有固定的增长率。$PV = \frac{C}{1+r}+\frac{C(1+g)}{(1+r)^2}+\frac{C(1+g)^2}{(1+r)^3}+...=\frac{C}{r-g}$



### 2.5. 名义利率与实际利率

+ 通货膨胀率（Inflation Rate）：物价上涨的速度
+ 名义利率（Nominal Interest Rate） ：投资价值增长的速度
+ 实际利率（Real Interest Rate） ：投资的购买力增长的速度
+ $1+实际利率=\frac{1+名义利率}{1+通货膨胀率}$，近似于$实际利率\approx 名义利率 -通货膨胀率$
  + 327国债事件——多头与空头较量
+ 考虑通货膨胀因素后的现值：一致性原则
  + 当现金流为“名义”现金流时，则贴现率为名义利率；
  + 当现金流为“实际”现金流时，则贴现率为实际利率；
